## Build Setup

### Features

- SPA (Single Page Application)
- SSR (Sever Side Rendering)

### Requires

Node.js v10.x
npm or Yarn

```
# install dependencies
$ npm run install # Or yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start
```
